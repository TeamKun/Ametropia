# Ametropia
近視/遠視クラフト  
遠くの視界や近くの視界がぼやけるようになる

 1.16.5 Forge

# 仕様
* ametropia  
    * セレクター
        * type
          * hyperopia 遠視
          * myopia 近視
          * none 無効化
        * level <0~1.0> 度数を設定
        * random プレイヤーごとにランダムに個人差を出す。
