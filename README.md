# Ametropia
近視/遠視クラフト  
遠くの視界や近くの視界がぼやけるようになる

 1.16.5 Forge

# 仕様
* ametropia  
    * セレクター
        * level 度数を設定(-で近視 +で遠視)
        * range <0~> 範囲を設定

度数付き眼鏡のgiveコマンド例  
/give @a ametropia:glasses{Dioptre:1.9f}
